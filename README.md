# Domestic Implementation Mapping — Policy Data Story

This repository contains a static HTML prototype for a policy-facing data story on domestic internalisation capacity in global health governance.

The project communicates a diagnostic-configurational mapping framework for Latin America. It is intended for PAHO/WHO-style audiences interested in IHR, Pandemic Agreement implementation, legal preparedness and differentiated country support.

The page should not present the model as causal inference, legal compliance assessment or mechanical forecasting. It should present the framework as an implementation-intelligence layer that helps identify country-specific bottlenecks and strategic entry points.
